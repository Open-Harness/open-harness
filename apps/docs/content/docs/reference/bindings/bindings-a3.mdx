title: Bindings (A3)

---

Bindings use the A3 (Access, Apply, Aggregate) grammar to connect data between nodes.

## Basic Bindings

Reference node outputs in other nodes:

```yaml
nodes:
  - id: node1
    type: generate
  - id: node2
    type: process
    input:
      data: "{{ node1.output }}"
```

## Binding Syntax

- `{{ nodeId.output }}` - Access a node's output
- `{{ nodeId.output.field }}` - Access a nested field
- `{{ input.field }}` - Access flow inputs
- `{{ nodeId.output | transform }}` - Apply transformations
