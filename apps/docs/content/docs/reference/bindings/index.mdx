# Bindings Reference

The A3 binding syntax allows nodes to reference flow inputs and other node outputs.

## Quick Reference

### Basic Syntax

```
{{path.to.value}}
```

### Flow Input Reference

```yaml
input:
  message: "{{flow.input.greeting}}"
```

### Node Output Reference

```yaml
input:
  data: "{{step1.output}}"
```

### Nested Paths

```yaml
input:
  name: "{{flow.input.user.profile.name}}"
  item: "{{step1.output.items.0.id}}"
```

### Optional Bindings

```yaml
input:
  # Empty string if not found
  optional: "{{?flow.input.maybeValue}}"
```

### Default Values

```yaml
input:
  # Use default if not found
  name: "{{flow.input.name | default:\"Anonymous\"}}"
  count: "{{flow.input.count | default:0}}"
```
