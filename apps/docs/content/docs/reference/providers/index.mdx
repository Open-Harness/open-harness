# Provider Reference

Providers are AI model backends that power agent nodes. They handle LLM communication, tool execution, and streaming.

## Available Providers

| Provider | Status | Description |
|----------|--------|-------------|
| [Claude SDK](/docs/reference/providers/claude-sdk) | Current | Anthropic Claude via Agent SDK |
| [OpenCode](/docs/reference/providers/opencode) | Experimental | OpenCode AI backend |

## Provider vs Channel

- **Providers** execute agent nodes (LLM calls, tool use)
- **Channels** handle I/O (console, voice, web)

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   Channel   │ ←→  │     Hub      │ ←→  │   Provider  │
│  (console)  │     │  (events)    │     │  (Claude)   │
└─────────────┘     └──────────────┘     └─────────────┘
```

## Agent Node Configuration

Agent nodes specify provider configuration:

```yaml
nodes:
  - id: analyze
    type: agent.claude
    input:
      prompt: "Analyze this data"
    config:
      model: claude-sonnet-4-20250514
      maxTurns: 3
```
