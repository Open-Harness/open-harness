## Quick Reference

### Flow Metadata

```yaml
flow:
  name: my-flow
  version: 1
  description: Optional description
  input:
    key: default_value
  nodePacks:
    - "@open-harness/core-pack"
  policy:
    failFast: true
```

### Node Policy

```yaml
nodes:
  - id: step1
    type: http
    input: {}
    policy:
      timeoutMs: 30000
      retry:
        maxAttempts: 3
        backoffMs: 1000
      continueOnError: false
```

### When Expressions

```yaml
nodes:
  - id: conditional
    type: process
    when:
      equals:
        var: check.output.valid
        value: true
    input:
      data: "{{check.output.result}}"
```
