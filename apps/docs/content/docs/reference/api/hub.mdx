The Hub is the unified bidirectional event bus that connects all components in Open Harness.

## Overview

The Hub provides:
- Event subscription and observation
- Command sending (`send`, `sendTo`, `sendToRun`, `reply`, `abort`)
- Context propagation (`scoped`, `current`)
