# Events

All communication in Open Harness flows through typed events.

## Event Structure

```typescript
interface RuntimeEvent {
  type: string;
  timestamp?: Date;
  runId?: string;
  nodeId?: string;
  [key: string]: unknown;
}
```

## Flow Events

### flow:start

```typescript
{
  type: "flow:start",
  flowId: "greeting-flow",
  runId: "run-123",
  input: { name: "World" }
}
```

### flow:complete

```typescript
{
  type: "flow:complete",
  flowId: "greeting-flow",
  runId: "run-123",
  output: { text: "Hello, World!" }
}
```

### flow:error

```typescript
{
  type: "flow:error",
  flowId: "greeting-flow",
  runId: "run-123",
  error: { message: "Something went wrong" }
}
```

## Agent Events

### agent:start

```typescript
{
  type: "agent:start",
  agentName: "claude",
  runId: "run-123"
}
```

### agent:text

Streaming text chunks:

```typescript
{
  type: "agent:text",
  content: "Hello",
  runId: "run-123"
}
```

### agent:tool_use

Tool invocation:

```typescript
{
  type: "agent:tool_use",
  name: "search",
  input: { query: "..." },
  runId: "run-123"
}
```

### agent:complete

```typescript
{
  type: "agent:complete",
  output: { text: "Final response" },
  runId: "run-123"
}
```

### agent:error

```typescript
{
  type: "agent:error",
  error: { message: "Rate limited" },
  runId: "run-123"
}
```

## Subscribing to Events

```typescript
// Specific type
hub.subscribe("agent:text", handler);

// Wildcard
hub.subscribe("agent:*", handler);

// All events
hub.subscribe("*", handler);
```
