# API Reference

The Open Harness kernel exports functions and classes for building AI-powered workflows.

## Core Functions

| Function | Purpose |
|----------|---------|
| [defineHarness](/reference/api/define-harness) | Create a harness factory for workflow execution |
| [createHub](/reference/api/hub) | Create a standalone event bus |
| [executeFlow](/reference/api/execute-flow) | Execute a YAML-defined flow DAG |

## Flow Functions

| Function | Purpose |
|----------|---------|
| [parseFlowYaml](/reference/api/parse-flow) | Parse YAML string into FlowYaml structure |
| [compileFlow](/reference/api/compile-flow) | Validate and compile flow for execution |
| [resolveBindings](/reference/api/resolve-bindings) | Resolve A3 bindings in templates |
| [evaluateWhen](/reference/api/evaluate-when) | Evaluate conditional expressions |

## Classes

| Class | Purpose |
|-------|---------|
| [NodeRegistry](/reference/api/node-registry) | Registry for custom node type definitions |

## Agent Providers

| Function | Purpose |
|----------|---------|
| [createAnthropicTextAgent](/reference/api/anthropic-agent) | Create Anthropic Claude text agent |

## Import

```typescript
import {
  defineHarness,
  createHub,
  executeFlow,
  parseFlowYaml,
  compileFlow,
  NodeRegistry,
  resolveBindings,
  evaluateWhen,
  createAnthropicTextAgent,
} from "@open-harness/kernel";
```
