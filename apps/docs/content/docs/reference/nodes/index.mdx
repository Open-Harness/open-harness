## Node Structure

Every node in a flow has:

```yaml
- id: unique-identifier
  type: category.nodeName    # e.g., control.foreach, agent.claude
  input:
    key: value               # Input bindings
  when:                      # Optional: conditional execution
    equals:
      var: someCondition
      value: true
  policy:                    # Optional: timeout, retry
    timeout: 30000
    retry: 3
```
