# Data Nodes

Data nodes handle constants, transformations, and data manipulation within flows.

## Available Data Nodes

| Node | Status | Description |
|------|--------|-------------|
| `constant` | Implemented | Output a static value |
| `transform` | Implemented | Transform data with expressions |

## Example: Constant

```yaml
nodes:
  - id: config
    type: constant
    input:
      value:
        apiUrl: "https://api.example.com"
        timeout: 30000
```

## Example: Transform

```yaml
nodes:
  - id: format-output
    type: transform
    input:
      data: "{{previous.output}}"
      template: |
        # Report
        Date: {{data.date}}
        Summary: {{data.summary}}
```
