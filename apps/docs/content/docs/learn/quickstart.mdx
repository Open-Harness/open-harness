# Quickstart

Build and run a "Hello World" flow in under five minutes.

## What You'll Build

A single-agent flow that takes a name and returns a greeting:

```
┌────────────────┐
│  hello.agent   │
│  (custom node) │
└────────────────┘
```

## Prerequisites

- Bun 1.0+
- A terminal with access to your project directory

## Step 1: Create a Project

```bash
mkdir my-first-flow
cd my-first-flow
bun init -y
bun add @open-harness/sdk
```

## Step 2: Define the Flow

Create `flow.yaml`:

```yaml
name: hello-world
nodes:
  - id: hello
    type: hello.agent
    input:
      name: "{{ flow.input.name }}"
edges: []
```

## Step 3: Create the Runtime

Create `run.ts`:

```typescript
import {
  createRuntime,
  DefaultNodeRegistry,
  parseFlowYaml,
  type NodeTypeDefinition,
} from "@open-harness/sdk";
import { readFileSync } from "node:fs";

const helloAgent: NodeTypeDefinition<{ name: string }, { message: string }> = {
  type: "hello.agent",
  run: async (_ctx, input) => ({ message: `Hello, ${input.name}!` }),
};

const flow = parseFlowYaml(readFileSync("flow.yaml", "utf-8"));

const registry = new DefaultNodeRegistry();
registry.register(helloAgent);

const runtime = createRuntime({ flow, registry });

runtime.onEvent((event) => {
  if (event.type === "node:complete" && event.nodeId === "hello") {
    console.log("Agent output:", event.output);
  }
});

const snapshot = await runtime.run({ name: "World" });
console.log("Final outputs:", snapshot.outputs);
```

Run it:

```bash
bun run run.ts
```

You should see output similar to:

```
Agent output: { message: "Hello, World!" }
Final outputs: { hello: { message: "Hello, World!" } }
```

## What Just Happened?

1. You defined a flow with a single custom agent node.
2. You registered the agent in a node registry.
3. You created a runtime and ran the flow with input.
4. You listened for runtime events and printed the output.
