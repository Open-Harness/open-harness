---
title: Flow Inputs
description: Pass data into flows and use inputs in nodes
---

Flows can accept inputs that are passed to nodes via bindings.

## Defining Inputs

Inputs are defined at the flow level:

```yaml
flow:
  name: example
  input:
    userName: string
    count: number
  nodes:
    - id: process
      type: my-node
      input:
        name: "{{ input.userName }}"
        iterations: "{{ input.count }}"
```

## Using Inputs in Nodes

Use the A3 binding syntax to reference flow inputs:

- `{{ input.fieldName }}` - Access a flow input field
- `{{ input.fieldName | default: "value" }}` - Provide a default value
