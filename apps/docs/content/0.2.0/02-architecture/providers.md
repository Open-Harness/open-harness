## Overview

Agent SDKs integrated with Open Harness. Swap/mix without code changes.

### Swapping Providers

- How to swap (change `type` in workflow YAML)
- No code changes required
- Same skills, different provider
- Compare performance (evals)

### Mixing Providers

- Different agents use different providers
- Claude for coding, OpenCode for analysis
- Codex for reasoning
- Skills work with all providers
