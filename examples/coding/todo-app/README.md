# TODO Application

A clean, modern TODO list application built with vanilla HTML, CSS, and JavaScript.

## Features

- ✅ **Add new todo items** - Quickly add tasks using the input field
- ✅ **Mark items as complete** - Check off tasks as you complete them
- ✅ **Delete items** - Remove tasks you no longer need
- ✅ **Clear completed** - Bulk remove all completed tasks
- ✅ **Task counter** - See how many tasks you have active
- ✅ **Empty state** - Helpful message when your list is empty
- ✅ **Responsive design** - Works great on desktop and mobile

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

```bash
# Open in default browser (macOS)
open index.html

# Open in default browser (Linux)
xdg-open index.html

# Open in default browser (Windows)
start index.html
```
