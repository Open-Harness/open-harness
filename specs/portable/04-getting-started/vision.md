# Vision

**Status:** Outline Only  
**Purpose:** Platform vision and long-term goals

---

## Overview

Open Harness is an open source platform for building agentic systems.

---

## Sections

### What Open Harness Is

- Platform for creating agentic systems
- Skills + Scripts + Evals pattern
- DX-first (irreducible internal, zero external)
- Production-grade from day one

### What Open Harness Is Not

- Not another LLM wrapper
- Not just coding workflows
- Not a toy framework
- Not a comparison to others

### The Agentic Singularity

- Happening now (not in the future)
- Models + SDKs getting better
- Capabilities increasing
- Your job = create agentic systems

### Long-Term Goals

- Open source core forever
- Ecosystem of skills, components, providers
- Anyone can build agentic systems
- Human flourishing (agentic capabilities for all)

### Open Source Forever

- Core platform = open source
- Nice license
- Products can be proprietary
- Monetization = ecosystem success
