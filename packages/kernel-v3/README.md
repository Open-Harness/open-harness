# kernel-v3

Graph-first workflow runtime for Open Harness (V3). This package contains only the
runtime, compiler, and core types. UI, servers, and adapters live elsewhere.

## Docs
- `docs/architecture.md`
- `docs/manifest.md`

## Install
```bash
bun install
```

## Scripts
```bash
bun run typecheck
bun run lint
bun run test
```
